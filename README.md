# media_mcp

音视频处理与合成方向的工程：**四阶段文档**位于 `ai-software-engineering/`，**Vue 实现**位于 `media-studio-vue/`（SDI、Windows 风格主框架）。

## 快速开始

```bash
cd media-studio-vue
npm install
npm run dev
```

浏览器打开终端提示的本地地址（默认 `http://localhost:5173`）。

## 文档

- 概念 / 逻辑 / 物理 / 运维：`ai-software-engineering/`
- 物理规格（实现权威）：`ai-software-engineering/02-physical/media-studio-vue/spec.md`
