# 产品设计：Media Studio（音视频合成）

## 愿景与范围

本仓库目标为交付一款 **单文档界面（SDI）** 的 **音视频处理与合成** 工作台：用户在 **单一主窗口** 内管理当前编辑会话（时间线/合成视图），通过 **Windows 经典纵向分区**（菜单栏、工具栏、客户区、状态栏）与 **可停靠面板** 完成素材管理与属性调整。

初期范围以 **壳层与交互基线** 为主：多语言、主题、状态栏与 Log、Dock 布局；媒体解码/编码与合成管线在后续迭代按物理规格扩展。

## 壳层类型：SDI

- **SDI（单文档界面）**：一个顶层主窗口承载当前工作区；一次聚焦一份合成会话（一个项目/时间线）。
- **新开会话**：可实现为同一窗口内切换项目，或再开顶层窗口（产品阶段二选一；当前为 **浏览器内单页**，通过菜单「新建项目」等在 **同窗口** 重置会话状态）。
- **关闭最后一个项目**：可提示是否退出或回到空状态（由实现与运维文档约定默认值）。

## Windows 风格界面（纵向分区）

| 区域 | 说明 |
|------|------|
| 菜单栏 | 文件/编辑/视图/帮助等；**必须**含「多语言」「主题」顶层菜单（见窗口规则） |
| 工具栏 | 高频操作图标为主，位于菜单栏下方 |
| 客户区 | 中央主工作区：以 **预览/合成画布** 为主；**时间线** 置于 **底部 Dock**（与左右 Dock 一致可折叠） |
| 状态栏 | 最下行；**可点击**打开 **Log 窗口**（纯文本历史、可复制全文） |

## Dock（停靠区）

- 主窗口内 **左/右/下** 可停靠 **媒体库、属性/检查器、时间线** 等面板。
- 支持 **展开/折叠**；同侧多面板时采用 **缘条（贴边折叠控件）+ 显示区（展开内容）** 的兄弟布局语义（与实现 `mapping.md` 对齐）。
