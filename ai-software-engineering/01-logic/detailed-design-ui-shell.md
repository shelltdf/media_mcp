# 详细设计：UI 壳（SDI + Windows 风格）

## 主窗口结构

1. **菜单栏**：文件（新建/打开/保存占位）、编辑、视图（切换 Dock）、**多语言**、**主题**、帮助。
2. **工具栏**：与「新建时间线」「导入媒体」等高频操作绑定（图标 + `title` 悬停说明）。
3. **客户区中部**：主 **预览** 区（时间线不在此区域）。
4. **Dock**：左侧「媒体库」、右侧「属性」、**底部「时间线」**；左右侧为纵缘条 + 显示区；底部为 **显示区在上、横缘条贴窗口最下**（与规则「下 Dock」一致）；各 Dock 可折叠。**标题栏**含 **折叠**、**最大化**（还原）按钮；Dock 与客户区/主行之间设 **可拖动分割条** 调整宽度或高度（`useDockLayout`）。
5. **状态栏**：单行摘要（最近一条日志）；**整块可点击** 打开 Log 窗口。

## Log 窗口

- 展示 **提示 / 警告 / 错误** 等级别（可扩展「信息」）。
- 正文为 **纯文本**（`<pre>` 或等价）；提供 **复制** 按钮复制 **当前 Log 全文**。
- 打开方式：状态栏点击。

## 语言与主题

- **语言**：中文、英文；切换后立即生效（`vue-i18n`）。
- **主题**：默认 **跟随系统**（`prefers-color-scheme`）；菜单提供浅色/深色/跟随。

## 典型流程

- 启动 → 显示空项目或上次会话（后续持久化）→ 用户导入媒体 → 日志记录 → 拖拽到时间线（后续）。
