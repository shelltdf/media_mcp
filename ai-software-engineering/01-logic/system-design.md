# 系统设计

## 子系统

| 子系统 | 说明 | 物理目标 |
|--------|------|----------|
| UI 壳 | SDI 主窗口、菜单/工具栏/状态栏/Dock/Log | `media-studio-vue` |
| 媒体与合成（占位） | 导入、时间线、预览、导出 | 同上，管线细节见详细设计 |

## 依赖关系

- UI 壳 **依赖** 统一日志与主题/语言服务（前端组合式函数或轻量 store）。
- 媒体管线 **依赖** 浏览器能力与后续原生/ WASM 扩展；当前不强制后端。

## 一致性

概念层 `product-design.md` 中 SDI 与 Windows 分区约束，在本阶段落实为 `detailed-design-ui-shell.md` 中的交互与状态流。
