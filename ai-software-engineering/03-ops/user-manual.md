# 用户说明：Media Studio

## 环境前提

- 现代浏览器（Chromium / Firefox / Edge 近期版本）。
- 开发预览：本机安装 **Node.js LTS**（见开发者手册）。

## 启动

- 开发：在 `media-studio-vue` 目录执行 `npm run dev` 或使用仓库提供的 `run.py` / `dev.py`，浏览器访问终端提示的本地 URL（默认 `http://localhost:5173`）。

## 界面概要（SDI）

- **单主窗口**：中央 **客户区** 为 **预览**；**时间线** 在 **底部 Dock**（可折叠横缘条）；两侧为可折叠 **媒体库**、**属性** Dock。各 Dock **标题栏** 有 **折叠**、**最大化/还原**；**分割条**（客户区与 Dock 之间、主行与时间线之间）可 **拖动** 调整宽度或高度。
- **属性与预览**：选中素材后，程序会 **串行** 使用 **ffmpeg.wasm**（与「转换为」共用同一实例、同一队列）对音视频做 **探针**（时长、分辨率、采样率、编码等，见属性面板「ffmpeg 探针」）以及 **预览转码**（生成 H.264/AAC 的短时预览片段供 `<video>` / `<audio>` 播放）。超大文件会 **跳过** 探针与预览转码（见界面提示），此时仍显示 **浏览器元数据** 与 **原始 blob URL** 预览。**通道列表** 仍以启发式推断为主；探针结果与浏览器读数 **优先合并展示**（时长/分辨率等）。无声音多为 **自动播放策略**（需用户点击播放或音量）。
- **媒体库**：可 **导入** 多文件（类型过滤见 `mediaAccept.ts`）。**单击** 选中素材，右侧 **属性** 显示详情与 **推断的数据通道**。**双击** 在 **预览区** 打开（视频/音频会尽量 **自动播放**；视频 **可带声音**，若浏览器拦截自动播放请用播放器控件 **播放/取消静音**）。**右键** 菜单：**预览**、**原始导出**、**转换为…**（须 **选择保存位置** 指定输出文件后再开始；在浏览器内通过 **ffmpeg.wasm** 转码，**首次**转换需联网下载编码核心；过程显示 **进度条**，结束后有 **完成** 提示并记入日志）、**复制文件名**、**删除**。**拖拽** 到 **时间线** 时按通道拆成多轨片段。
- **菜单栏**：含 **多语言**（中文/英文）、**主题**（浅色/深色/跟随系统）。
- **状态栏**：显示最近一条提示；**点击状态栏** 打开 **Log 窗口**，可查看全部历史并 **复制全文**。

## 常见问题

- **主题未随系统变化**：请确认菜单中主题为「跟随系统」；部分浏览器需刷新页面后首次识别 `prefers-color-scheme`。
