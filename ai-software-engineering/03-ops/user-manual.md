# 用户说明：Media Studio

## 环境前提

- 现代浏览器（Chromium / Firefox / Edge 近期版本）。
- 开发预览：本机安装 **Node.js LTS**（见开发者手册）。

## 启动

- 开发：在 `media-studio-vue` 目录执行 `npm run dev` 或使用仓库提供的 `run.py` / `dev.py`，浏览器访问终端提示的本地 URL（默认 `http://localhost:5173`）。

## 界面概要（SDI）

- **单主窗口**：中央 **客户区** 为 **预览**；**时间线** 在 **底部 Dock**（可折叠横缘条）；两侧为可折叠 **媒体库**、**属性** Dock。各 Dock **标题栏** 有 **折叠**、**最大化/还原**；**分割条**（客户区与 Dock 之间、主行与时间线之间）可 **拖动** 调整宽度或高度。
- **媒体库**：可 **导入** 多文件（类型过滤见 `mediaAccept.ts`）。**单击** 选中素材，右侧 **属性** 显示详情与 **推断的数据通道**（视频轨、音轨、字幕等占位模型）。**双击** 在 **预览区** 打开。**拖拽** 到下方 **时间线** 时，按通道拆成多轨片段（视频 / 音频 / 字幕 / 数据 四行）。
- **菜单栏**：含 **多语言**（中文/英文）、**主题**（浅色/深色/跟随系统）。
- **状态栏**：显示最近一条提示；**点击状态栏** 打开 **Log 窗口**，可查看全部历史并 **复制全文**。

## 常见问题

- **主题未随系统变化**：请确认菜单中主题为「跟随系统」；部分浏览器需刷新页面后首次识别 `prefers-color-scheme`。
