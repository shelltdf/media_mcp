# Media Studio（Vue）

基于 Vue 3 + Vite 的 **SDI** 音视频合成工作台壳层：Windows 风格菜单栏 / 工具栏 / 客户区 / 状态栏、左右 Dock、状态栏打开 Log、中英文与主题（含跟随系统）。

## 命令

| 命令 | 说明 |
|------|------|
| `npm install` | 安装依赖 |
| `npm run dev` | 开发服务器 |
| `npm run build` | 生产构建 → `dist/` |
| `npm run preview` | 本地预览构建结果 |
| `npm run test` | 类型检查 + 构建（冒烟） |

仓库根目录亦提供 `build.py`、`test.py`、`run.py`、`publish.py`、`dev.py` 封装上述流程。

## 环境

- Node.js LTS 20+ 推荐
